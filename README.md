# 🚀 Agency Dashboard

A modern, responsive **Agency Management Dashboard** built using **HTML, Tailwind CSS, and Vanilla JavaScript**. This project is designed to help digital agencies manage clients, projects, and workflows efficiently with a clean and professional interface.

---

## 🌐 Live Demo

👉 https://jakariaofficial.github.io/agency_dashboard/

---

## ✨ Features

- 📊 **Dashboard Overview** – Clean and structured layout  
- 👥 **Client Management** – Add clients dynamically via modal  
- 📁 **Project Tracking UI** – Organized sections for scalability  
- 🔐 **Frontend Authentication** – LocalStorage-based login system  
- 📱 **Responsive Design** – Mobile-first approach  
- 🎨 **Modern UI/UX** – Tailwind CSS + smooth animations  
- 🔔 **Custom Toast Notifications**  
- 🧩 **Reusable Layout** – Sidebar, header, and content separation  

---

## 🛠️ Tech Stack

- **HTML5** – Semantic structure  
- **Tailwind CSS** – Utility-first styling  
- **Vanilla JavaScript** – DOM manipulation & logic  
- **Font Awesome** – Icons
